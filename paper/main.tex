\begin{abstract}
We introduce a computational framework for analyzing complex systems using topological invariants derived from phase fields. We define $Q$ and $Q_{abs}$ and test them on synthetic fields, 3D defect dynamics, physics snapshots, and EEG pipelines with PCI validation hooks.
\end{abstract}
