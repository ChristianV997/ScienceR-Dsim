# Book Seed Q3 — Soltar sin creer (Budismo para escépticos, ciencia-pop) v0.1

## Intención

Un libro para gente **escéptica, racional y joven**, que solo confía en ciencia: explicar soltar/liberación con lenguaje común, analogías pop y conceptos de teoría de información (sin volverse cringe).

## Promesa (1 línea)

Puedes reducir sufrimiento aprendiendo a **descomprimir tu mente**: menos ruido, menos loops, mejores decisiones bajo emoción.

## Lector ideal

- Le interesa psicología, neuro, productividad.
- Odia dogmas; ama explicaciones claras.

## Traducción (budismo → lenguaje moderno)

- vedanā → "señal" (pleasant/unpleasant/neutral)
- taṇhā → "impulso" / "urgencia"
- upādāna → "pegajosidad" / "sobre-identificación"
- dukkha → "fricción" (dolor + resistencia)
- samsara → "ciclo de loops" (repetición por aprendizaje mal calibrado)

## Modelos principales (ciencia-pop)

1. **Loop de predicción**: el cerebro intenta minimizar sorpresa, pero a veces se vuelve adicto a controlar.
2. **Ganancia/volumen**: cuando el sistema nervioso sube sensibilidad y todo se siente "más".
3. **Compresión vs sobreajuste**: soltar es dejar de sobreajustar historias al dolor.
4. **Latencia a la reacción**: libertad = aumentar 1–3 segundos entre señal y respuesta.

## Estructura (partes)

1. **Por qué tu mente se pega** (sin culpa)
2. **La física del impulso** (cuerpo + atención)
3. **La segunda flecha** (cómo tú mismo subes la fricción)
4. **Samsara como algoritmo** (repetición + refuerzo)
5. **Soltar como habilidad** (prácticas cortas)
6. **Relaciones y redes sociales** (dopamina, comparación, narrativa)
7. **Muerte y perspectiva** (sin religión)

## Prácticas base (versión "ciencia + calle")

- **Protocolo 90 segundos**: sentir sin actuar, observar curva.
- **3 respiraciones**: bajar ruido, subir señal.
- **Etiqueta mínima**: "aparece urgencia / aparece miedo".
- **Reducción de bitrate**: menos inputs cuando estás reactivo.

## Seeds (capítulos semilla)

### Seed — Tu mente es un feed: no todo merece click

La urgencia es un anuncio. No significa que sea verdad. Si no haces click, se va.

### Seed — El miedo es un algoritmo que exagera

El miedo se entrena para sobrevivir, no para ser exacto. Tu práctica es actualizar el modelo con calma.

### Seed — Samsara como loops recursivos

Samsara no es un lugar; es un patrón. El mismo loop: estímulo → impulso → historia → sufrimiento → repetición. Salir no requiere creer nada. Requiere interrumpir el loop en cualquier punto. La interrupción más efectiva: entre señal y respuesta. Eso es lo que entrenas.

## Guardrails

- No prometer "felicidad permanente".
- No atacar religión; solo ofrecer una lectura útil.
- El budismo aparece como "ingeniería de la mente" con respeto.
