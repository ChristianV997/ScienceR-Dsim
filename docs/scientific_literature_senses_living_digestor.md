# Scientific Literature Senses & Living Digestor (P37)

Purpose: claim-governed fixture-mode literature sensing pipeline.

This follows TOE/P34/P35 by adding registries, query packs, retrieval planning, fixture retrieval, normalization, dedupe, scoring, claim extraction, evidence tiering, construct mapping, falsifier watchlist, synthesis, validator, Obsidian sync, command-center payloads, and RAG pack.

Guardrails: no live API calls by default, no OpenAI/API keys required, no diagnosis/treatment/proof/ontology-promotion claims.

How to run:
- `make literature-senses-plan`
- `make literature-senses-fixture-run`
- `make validate-literature-senses`
- `make literature-senses-sync-obsidian`
- `make literature-senses-command-center-payloads`
- `make literature-senses-rag-pack`
- `make literature-senses-cycle`
