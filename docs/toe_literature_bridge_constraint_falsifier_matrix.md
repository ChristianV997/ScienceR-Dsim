# TOE Literature Bridge and Constraint/Falsifier Matrix

## 1. Purpose

P40 converts existing research artifacts into a traceable, falsifier-driven theory-integration layer. All outputs remain provisional research scaffolding.

## 2. Why P40 follows P37, P37.1, and P39

P37 supplies external literature-sense artifacts, P37.1 summarizes the internal project corpus, and P39 records reusable public-repository patterns. P40 maps those inputs into explicit constructs, constraints, equations, and failure conditions.

## 3. Inputs consumed

The generator inspects `outputs/literature_senses`, `outputs/project_corpus_digest`, `outputs/public_repo_harvest`, `outputs/tol_digest`, and `outputs/tol_digest/mental_health_bridge`. Missing roots are recorded as `not_available`; `--strict` additionally returns a failure status.

## 4. TOE primitive expansion

The priority matrix covers boundaries, constraints, fields, topology, information, symmetry, phase transitions, observer/interiority, viability, allostatic cost, precision weighting, recalibration latency, local-global consistency, causal topology, cosmological field constraints, and the GR/Kerr reduction limit.

## 5. Topology telemetry upgrade

The topology digest records Q, Qabs, fdress, Hodge and persistent local Laplacians, sheaf consistency, directed flow, causal topology, and local-global mismatch. These are candidate telemetry families subject to resolution, interpolation, and null controls.

## 6. Active inference and allostasis bridge

The active-inference digest maps free-energy objectives, precision weighting, allostatic cost, interoceptive/exteroceptive arbitration, and recalibration latency into testable model language.

## 7. Computational psychiatry bridge

The computational-psychiatry digest tracks latent parameters, attractor dwell, recovery latency, threat precision, exit cost, and self-model recurrence. It is non-diagnostic and does not prescribe intervention.

## 8. Bioelectric and basal cognition bridge

The bioelectric digest separates adaptive boundary regulation, multiscale autonomy, non-neural anticipation, and substrate-independent regulation from unsupported subjective-experience conclusions.

## 9. Cosmology constraint matrix

The cosmology matrix covers the standard cosmological model, BAO, DESI, CMB, SNe, Euclid lensing, the Hubble tension, and dark-energy equation-of-state constraints. Candidate deviations must be fitted against joint observational bounds.

## 10. Gravitational-wave constraint matrix

The gravitational-wave matrix covers Kerr ringdown, quasi-normal modes, black-hole spectroscopy, the area theorem, and the GR reduction limit. Candidate equations must reduce appropriately or be rejected.

## 11. Consciousness adversarial matrix

The adversarial matrix compares BTC-ICFT telemetry with Global Workspace Theory, Integrated Information Theory, Recurrent Processing Theory, Active Inference, Higher-Order Theories, Orch-OR, and Predictive Processing. Each row records shared and divergent predictions plus a required experiment.

## 12. Falsifier registry

`toe_falsifier_watchlist.json` defines F1-F10 with the affected claim, subsystems, required observable, failure condition, demotion action, next experiment, and priority. The registry spans neural residual value, anesthesia, interpolation, topology controls, richer topology families, longitudinal flexibility, cosmology, ringdown, bioelectric interpretation, and mechanism replication.

## 13. Command-center integration

The command-center payload generator publishes registry summaries while forcing real-data auto-execution, live APIs, key exposure, clinical claims, and ontology promotion off.

## 14. RAG integration

The RAG pack exports safe registry summaries, chunking guidance, query examples, blocked answer patterns, and public-answer guidance. It preserves hypothesis scope and explicit uncertainty.

## 15. Guardrails

P40 performs no network calls, dataset downloads, real-data execution, label inference, target fabrication, or secret handling. Unsafe certainty, clinical, and ontology language is rejected by the validator.

## 16. What this does not claim

The bridge does not establish metaphysical identities, subjective experience, clinical efficacy, or empirical support from fixture artifacts. Physics matrices constrain candidate models; they are not evidence for them.

## 17. How to run

```bash
make toe-literature-bridge
make toe-literature-bridge-topology
make toe-literature-bridge-constraints
make validate-toe-literature-bridge
make toe-literature-bridge-command-center-payloads
make toe-literature-bridge-rag-pack
make toe-literature-bridge-sync-obsidian
make toe-literature-bridge-cycle
```

For an offline Python-only run, invoke the modules listed by those targets directly under `tools.toe_research.literature_bridge`.
